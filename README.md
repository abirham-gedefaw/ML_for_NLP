# 🤖 Machine Learning for Natural Language Processing (NLP)

A comprehensive documentation log and code repository tracking my hands-on journey through Machine Learning applied to Natural Language Processing.

## 📌 Project Overview
This repository serves as my personal lab notebook for learning and implementing NLP techniques. It documents everything from basic text preprocessing to advanced model architectures.

## 🛠️ Core Stack & Tools
* **Language:** Python 3.x
* **Environment:** VS Code / Anaconda
* **Libraries:** NLTK, NumPy, Pandas

## 📖 What I've Covered So Far

### 1. Text Preprocessing Foundations
* **Environment Setup:** Verifying NLTK installation inside active Anaconda environments.
* **Tokenization:** 
  * Sentence splitting using `sent_tokenize` and the NLTK `punkt_tab` model.
  * Word segmentation utilizing standard `word_tokenize` vs. specific syntactic rules of the `TreebankWordTokenizer`.

---

## 🚀 How to Run the Scripts Locally

### Prerequisites
Make sure you are using the correct Python interpreter linked to your Anaconda environment.

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   cd ML_for_NLP
   ```

2. **Verify NLTK installation:**
   ```bash
   pip show nltk
   ```

3. **Run a test tokenization script:**
   ```bash
   python test_script.py
   ```
